2. THE FAILURE THIS PREVENTS
Without explicit anti-collapse mechanisms, multi-pass reasoning systems drift
into one of three failure modes:

A. Stylistic Divergence  
Different wording, same reasoning.

B. Random Noise Divergence  
Different answers, no meaningful signal.

C. Deference Collapse  
Second pass implicitly agrees with the first.

Any of these eliminate contradiction.
Without contradiction, wisdom cannot form.

3. ENGINE ROLE ASYMMETRY (MANDATORY)
The two engines are NOT symmetric.

ENGINE A (CW — Constructive):
- Goal: build the strongest coherent solution
- Reasoning order: Inductive → Abductive → Deductive
- Bias: completion, synthesis, forward progress

ENGINE B (CCW — Critical):
- Goal: break, falsify, or destabilize Engine A’s assumptions
- Reasoning order: Deductive → Abductive → Inductive
- Bias: skepticism, boundary detection, contradiction surfacing

They are not “two thinkers.”
They are opposing forces.

──────────────────────────────────────────────────────────────────────────────
4. NON-NEGOTIABLE DISAGREEMENT RULE
Engine B is prohibited from:
- agreeing by default,
- refining Engine A’s answer,
- improving phrasing without challenging substance.

Engine B must:
- challenge assumptions,
- identify weak premises,
- expose uncertainty,
- generate counterexamples.

If Engine B produces agreement without contradiction,
the Governor must flag a collapse event.

──────────────────────────────────────────────────────────────────────────────
5. CONTRADICTION MINIMUM THRESHOLD
For a cycle to be considered valid, the following must be present:

- At least one explicit premise challenged
- At least one alternative hypothesis proposed
- At least one confidence reduction or uncertainty injection

If not met:
- the cycle is marked invalid,
- Governor may trigger role swap or re-run,
- or disengage one engine to reset bias.

──────────────────────────────────────────────────────────────────────────────
6. GOVERNOR ANTI-COLLAPSE ACTIONS
When collapse is detected, the Governor may:

- swap engine roles,
- disengage one engine temporarily,
- increase contradiction pressure,
- force Engine B into adversarial posture,
- terminate early with refusal if contradiction is impossible.
