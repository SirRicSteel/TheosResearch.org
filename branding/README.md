## Usage Guidelines

### ✅ Approved Uses
- Repository README header
- Executive Synthesis cover
- Pitch deck title slides
- Presentation materials
- Website branding
- Social media profiles

### ⚠️ Conditional Uses
- Technical specifications (small logo in header only)
- Formal documents (use sparingly)
- Academic papers (check journal guidelines)

### ❌ Do Not Use
- Code repositories (keep technical)
- Legal documents (use institutional branding)
- Any context that might imply consciousness claims

---

## Brand Messaging

### Core Tagline
> "Transparency is a governance choice."  
> — Frederick Davis Stalnecker

### Official Description
> THEOS™: A Governed Reasoning Architecture

### Key Positioning
- **What THEOS is:** A governance layer for AI systems
- **What THEOS does:** Enables consequence-based wisdom without consciousness
- **What THEOS provides:** Posture-based safety, functional time, cumulative wisdom

### What NOT to Say
- ❌ "AI Consciousness" (THEOS does not claim consciousness)
- ❌ "Self-aware AI" (THEOS is not sentient)
- ❌ "Memory-based learning" (use "wisdom state" instead)

## Logo Symbolism

**Eye of Providence** - Oversight, governance, transparency  
**Globe** - Global applicability, universal principles  
**Ionic Column** - Classical wisdom, stability, foundational architecture  
**Stars and Moon** - Timeless, celestial, enduring  
**Gold Color** - Premium quality, trustworthiness, value  
**Glass Cube** - Transparency (literal and metaphorical)
