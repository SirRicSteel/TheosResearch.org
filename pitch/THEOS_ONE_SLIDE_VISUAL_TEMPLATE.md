## DESIGN NOTES FOR VISUAL CONVERSION:

### Color Scheme:
- **Primary:** Gold (#D4AF37) - matches logo
- **Secondary:** Deep blue (#1a365d) - professional, tech
- **Accent:** Teal (#2c7a7b) - represents transparency
- **Background:** White or very light gray

### Typography:
- **Headings:** Bold, sans-serif (Helvetica Bold, Arial Bold)
- **Body:** Clean sans-serif (Helvetica, Arial)
- **Metrics:** Large, bold numbers (33%, 56%)

### Layout:
- **Top section:** Logo + tagline
- **Problem section:** Left side or top
- **Solution section:** Center - largest visual space
- **Evidence section:** Grid of 4 boxes with metrics
- **Opportunity section:** Horizontal flow diagram
- **Ask section:** Bottom with contact info

### Key Visual Elements:
1. **THEOS logo** at top (use IMG_3064.jpeg with tagline)
2. **Dual-engine diagram** in Solution section (show counter-rotation)
3. **Metric boxes** in Evidence section (use color to highlight)
4. **Integration diagram** in Opportunity section (+ symbol prominent)

### PowerPoint Conversion Steps:
1. Open PowerPoint, create blank slide
2. Set slide size to 16:9 widescreen
3. Add logo at top center
4. Create text boxes for each section
5. Add simple shapes (rectangles, arrows) for diagrams
6. Export as PDF when complete

### Canva Conversion Steps:
1. Go to canva.com
2. Search "Infographic" templates
3. Find clean, professional template
4. Replace text with this content
5. Adjust colors to match THEOS branding
6. Download as PNG or PDF
